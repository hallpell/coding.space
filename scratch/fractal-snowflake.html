Fractal Snowflake

We're going to be making mathematical snowflakes!

The examples are going to be going slow to let you see what's happening, but you don't need to do this for your project.

1) Delete the cat sprite and replace it with a smaller sprite so you can see what you're drawing. If you'd like, you can remix the project below to use the same sprite as the tutorial

Sample Sprite

2) Use the pen extension to draw an equilateral triangle.

3) Now, instead of each side of the triangle, lets add a bend. Instead of drawing '___', lets draw '_/\_'.

Try using "Make a block" to define a block that you can replace your move block with.

4) Now, instead of only replacing each side of the triangle, lets replace every line we draw with a bend.

(Hint: Your block can use itself when defining what it does. You will need to make your block take a number as input (and use that number))

5) You can experiment with the block you defined to change your snowflake. Try changing the color of the pen, angles that you turn, or even what you replace each line with.

Wrap up: This is a fractal known as the Koch snowflake.

One of the cool things about it is that (at infinite depth), the line is infinitely long.

The process of a block using itself in its definition is called recursion, and can be used in many places.
